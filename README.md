# SIMILIS.AI — прототип (портфолио-мок)

Интерактивная реконструкция реального продукта SIMILIS.AI — визуального поиска археологических артефактов (ЕУ СПб + ИИМК РАН + НовГУ). Vite + React + Recharts, offline-first, GitHub Pages. По умолчанию русский, есть EN.

> **Данные реальные:** изображения и метаданные из проекта SIMILIS. Два владельца: **ИИМК РАН** (СПб: Ниеншанц/Охта 1, Большая Посадская, Нейшлотский, Тульская) и **НовГУ** (Старая Русса, Пески — из базы `portal.novsu.ru/archeology`). Similarity в демо — **офлайн-прокси** прод-эмбеддингов ResNet50/CLIP (иллюстрация, помечена). Прод-метрики — факт проекта, изолированы в Дашборде.

## Что внутри
- **Поиск:** выбор из 5 preset-примеров (загрузка своего фото — «Prod Only»), рабочие контролы — модель (ResNet50/CLIP), порог схожести, поставщик данных; сетка карточек с бейджем владельца и score; empty state; избранное; статичная SVG-карта СЗ России с пинами по памятникам (подсветка активных).
- **Карточка артефакта:** шифр, наименование, материал, описание, размер, датировка, квадрат, объект, раскоп, памятник, владелец.
- **Дашборд:** блок «Продакшен (факт)» (50k+ изображений, Recall@20 >90%, TTFR <1c, HITL <5%) + блок «Локальное демо» (счётчики + donut типов + bar по памятникам).

## Данные (data-prep, свой)
- `verify/data_prep.py` — джойн `Validation_dataset` (имя файла = инв. код) → `ARDB_Описи.xlsx`, ресайз картинок, визуальные признаки (цвет+форма), KNN-граф, PII-гейт. 82 объекта ИИМК РАН.
- `verify/merge_novgu.py` + `verify/novgu_finds.json` — 14 находок НовГУ (Старая Русса/Пески), хотлинк миниатюр НовГУ, similarity по материалу. Итог: 96 объектов, 2 владельца.
- `verify/embed_real.py` — **опционально, локально**: считает НАСТОЯЩИЕ ResNet50 + CLIP эмбеддинги (`pip install torch torchvision open_clip_torch`) и перезаписывает граф → плашка «real embeddings». В песочнице torch недоступен (прокси), поэтому запускать на маке.

## Запуск / сборка / деплой
```bash
npm install
npm run dev                 # http://localhost:5173
npm run build               # → dist/
GHPAGES_BASE=/<repo>/ npm run deploy   # GitHub Pages
```

## Честность
- Изображения/метаданные — реальные, право на публикацию подтверждено владельцем проекта.
- PII-гейт: пользовательские данные (email) в пайплайн не попадают (regex-ассерт в сборке).
- Similarity помечена (прокси/реальная); прод-метрики отделены от demo.
- Изображения НовГУ — хотлинк миниатюр их портала (с fallback). Для полного offline — скачать 14 миниатюр локально в `public/img/` и переключить `external:false` в `merge_novgu.py`.
